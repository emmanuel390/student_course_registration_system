## Project Description
The Student Course Registration System is a console-based application developed using Python.  
It allows users to add students, add courses, register students for courses, and display registered courses.

This project demonstrates the full Software Development Life Cycle (SDLC), from planning to deployment.

---

## Features
- Add a student
- Add a course
- Register a student for a course
- Display students and their registered courses

---

## SDLC Phases Used
1. Planning
2. Requirement Analysis
3. System Design
4. Implementation
5. Testing
6. Deployment and Maintenance
